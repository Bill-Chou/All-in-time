指標說明與風險提醒

策略邏輯

本儀表板基於「多頭 Regime 下的短線均值回歸」概念：在確認市場處於健康多頭環境（Regime Score ≥ 70）後，當短線指標出現極端超賣時，視為潛在的均值回歸進場機會。

指標說明

  • Regime Score：趨勢 + 斜率 + 波動 + 風險偏好（0–100）
  • RSI(2)：2 日 RSI，偵測短線極端超賣
  • RSI(2) 連續天數：連續低於門檻天數，越多反彈機率越高
  • IBS：(Close-Low)/(High-Low)，< 0.2 為弱勢收盤
  • SMA20 乖離率：偏離均線程度，< -3% 為明顯回撤

風險提醒

  • 本 Dashboard 僅供研究輔助，不構成投資建議
  • 均值回歸策略通常持有 3–5 天，不適合長期持有
  • 即使條件滿足，仍可能因黑天鵝事件造成損失
  • 請根據自身風險承受能力設定停損

資料來源

  • 數據源：Yahoo Finance chart API
  • 因 CORS 限制，建議透過 HTTP Server 或 GitHub Pages 執行
  • VIX / breadth proxy 為獨立數據源，可能偶爾失敗
