📐 策略邏輯

本儀表板基於「多頭 Regime 下的短線均值回歸」概念：在確認市場處於健康多頭環境（Regime Score ≥ 70）後，當短線技術指標出現極端超賣（RSI(2) < 15）時，視為潛在的均值回歸進場機會。

📊 指標說明

  • Regime Score：綜合趨勢位置、趨勢斜率、波動健康度、風險偏好四大面向（0–100 分）
  • RSI(2)：2 日相對強弱指標，用於偵測短線極端超賣
  • RSI(5)：5 日 RSI，作為輔助確認
  • Williams %R(2)：2 日威廉指標，< -80 確認超賣
  • VIX：CBOE 波動率指數，反映市場恐懼情緒
  • 20-day Realized Vol：過去 20 日的年化實現波動率

⚠️ 風險提醒

  • 本 Dashboard 僅供研究與決策輔助，不構成投資建議
  • 均值回歸策略通常持有天數偏短（3–5 天），不適合長期持有
  • 即使所有條件滿足，仍可能因黑天鵝事件造成損失
  • 請務必根據自身風險承受能力，設定適當的停損機制
  • 過去表現不代表未來結果

🔌 資料來源與限制

  • 數據來源：Yahoo Finance chart API
  • 因瀏覽器 CORS 限制，直接開啟 HTML 可能無法抓取數據
  • 建議透過本地 HTTP Server 或支援 CORS 的環境執行
  • VIX 與 breadth proxy（RSP/QQEW）為獨立數據源，可能偶爾取得失敗
